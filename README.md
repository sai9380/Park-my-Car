# 📖 Overview

**Park My Car** is a web-based Car Parking Management System developed to simplify parking space management for both administrators and users.

The system provides a secure and efficient way to manage parking slots, monitor vehicle records, calculate parking fees, and streamline the overall parking process through an easy-to-use web interface.

# ⚙ Installation Guide

### 1 Clone Repository

```bash
git clone https://github.com/sai9380/Park-my-Car.git
```

### 2 Move Project

Copy the project folder into

```
xampp/htdocs/
```

### 3 Start XAMPP

Start

- Apache
- MySQL

### 4 Import Database

Open

```
http://localhost/phpmyadmin
```

Create a database and import

```
database/carparkingsystem.sql
```

### 5 Configure Database

Update

```
config.php
```

if required.

### 6 Run the Project

```
http://localhost/carparkingsystem/
```
